# rewilding2023

Follow-up of Toby's master project on the effects of food-web context on the success of predator reintrodution.

The results of this work is published as non peer-reviewed preprint:

Thomas F. Johnson, Alain Danet, Toby Haefling, Andrew P. Beckerman (2025). Reintroductions backfire by destabilising food webs and triggering further extinction cascades.
bioRxiv 2025.08.28.672799; doi: https://doi.org/10.1101/2025.08.28.672799 

The food-web simulations were run using the Julia package [EcologicalNetworksDynamics.jl](https://github.com/econetoolbox/EcologicalNetworksDynamics.jl). Please look at the associated publication of [Lajaaiti et al. 2025](https://onlinelibrary.wiley.com/doi/abs/10.1111/2041-210X.14497) for more information.
